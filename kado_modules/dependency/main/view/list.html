Kado is built largely upon open source software that has been created as part of the Node.js revolution.

The primary Kado code is built and maintained by the Nullivex Web Consortium. Which follows the Kado Release Schedule.

Kado comprises of many open source dependencies that really make Kado great, and for that we say thank you! A non exhaustive list is provided below. If you see something missing Contact Us!

Donations

If you feel Kado has helped better your life in some way and you would like to help contribute to the group that helps maintain this code there are a couple ways to do it:

  • Create a Module
  • Create a Theme
  • Contribute a Feature or Fix
  • Donate

If you would simply like to donate please use the button at the bottom of the page.
